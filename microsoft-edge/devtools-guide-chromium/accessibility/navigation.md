<span data-ttu-id="99aed-277">Der Überwachungsbericht ist in Abschnitte gegliedert, die den einzelnen Überwachungskategorien entsprechen.</span><span class="sxs-lookup"><span data-stu-id="99aed-277">The audit report is organized into sections that correspond with each of the audit categories.</span></span>  <span data-ttu-id="99aed-278">Der Bericht wird mit einer Liste der Bewertungen für jede Kategorie geöffnet.</span><span class="sxs-lookup"><span data-stu-id="99aed-278">The report opens with a list of scores for each category.</span></span>  <span data-ttu-id="99aed-279">Diese Punkte sind auch Links, die verwendet werden können, um die entsprechenden Abschnitte zu überspringen.</span><span class="sxs-lookup"><span data-stu-id="99aed-279">These scores are also links which are able to be used to skip to the relevant sections.</span></span>  <span data-ttu-id="99aed-280">In jedem Abschnitt handelt es sich `details` um erweiterbare Elemente, die Informationen zu übergebenen oder fehlgeschlagenen Audits enthalten.</span><span class="sxs-lookup"><span data-stu-id="99aed-280">Within each section are expandable `details` elements, which contain information relating to passed or failed audits.</span></span>  <span data-ttu-id="99aed-281">Standardmäßig werden nur fehlerhafte Überwachungen angezeigt.</span><span class="sxs-lookup"><span data-stu-id="99aed-281">By default, only failing audits are shown.</span></span>  <span data-ttu-id="99aed-282">Jeder Abschnitt endet mit einem Final `details` -Element, das alle übergebenen Audits enthält.</span><span class="sxs-lookup"><span data-stu-id="99aed-282">Each section ends with a final `details` element which contains all of the passed audits.</span></span>
